# Проект по заданиям GUI C#

Этот репозиторий содержит решения трёх задач по C#.

## Задача 1 — Оконное приложение

- Проект **GeometryApp** представляет собой полное решение для задачи 1.
- Приложение реализует:
  - Отображение геометрических фигур (прямоугольник и круг)
  - Управление параметрами фигур: цвет, текст, размер, поворот, сжатие
  - Сохранение состояния фигур при переключении между ними
  - Интерфейс с ползунками, кнопками и полями ввода
- Архитектура построена по паттерну **MVVM** с использованием Avalonia.

## Задача 2 — Code Review

- Класс **HotelBooking** является решением для задачи 2.
- Реализует функциональность онлайн-бронирования номеров:
  - Регистрация гостей
  - Проверка доступности комнат
  - Сохранение бронирований в базу данных
- Особое внимание уделено:
  - Потокобезопасности при параллельных запросах
  - Использованию Dependency Injection для работы с базой данных и сервисом комнат

## Задача 3 — Вспомогательный метод

- Класс **SolutionTask3** является решением для задачи 3.
- Реализует метод-расширение `FilterAndCount`, который:
  - Принимает коллекцию элементов
  - Фильтрует её по условию
  - Возвращает количество элементов, удовлетворяющих фильтру
- Решение оптимизировано для работы с любыми коллекциями без лишнего создания промежуточных списков.
